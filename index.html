Front End Engineering Bootcamp Ruangguru x Kampus Merdeka

Februari, 16 Maret 2022

Awal mula saya ikutan program studi independent kampus merdeka ini hanya coba-coba dulu aja. Waktu itu saya sudah daftar di beberapa mitra. Namun tidak ada panggilan lagi. Sempet berfikir, mungkin tidak akan ada lagi panggilan dari mitra lain. Akhirnya di beberapa hari menjelang penutupan pendaftaran studi independent, ada lah notif di email dari Ruangguru. Karena memang saya daftar juga di mitra Ruangguru, yaitu daftar sebagai Front End Engineering Bootcamp. Disitu saya langsung senang, akhirnya bisa diterima jugaa.. Harapan ke depan nya dengan mengikuti program ini, semoga saya bisa mengembanngkan skill saya. Lalu menambah wawasan luas juga, menambah relasi, menambah pengalaman jugaa..
